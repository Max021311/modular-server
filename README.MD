# Server for modular project

## How to run the project

Install dependencies.
```sh
yarn
```

### Production mode

Build the project.
```sh
yarn build
```

Run the project.
```sh
yarn start
```

### Development mode

```sh
yarn dev
```


## Linter

This project use Eslint to lint and format the code.

For lint the project.
```sh
yarn lint
```

To fix basic errors and format.
```sh
yarn lint:fix
```
